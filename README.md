# Puzzleworks

A daily puzzle website where every puzzle — rules, code, and design — is invented by an LLM and published without human edits.

Once a day, a GitHub Action asks the model for a brand-new, self-contained browser puzzle, validates it in a headless browser, and publishes it to GitHub Pages. One puzzle per day, never a repeat.
